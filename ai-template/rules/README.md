# Rules Module

This module contains development rules used by AI agents and team members.

## Priority

1. Project-specific rules (filled by agent/user for this repository).
2. Generic baseline rules from this template.
3. Tool defaults (linters/framework defaults) if not overridden above.

## Fill Strategy

- Keep stable, universal rules prefilled.
- Keep project-dependent rules as guidance comments and fill them from project context.
