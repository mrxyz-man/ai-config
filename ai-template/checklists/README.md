# Checklists Module

This module stores reusable checklists for recurring delivery and review workflows.

How to use:
- Keep checklist items action-oriented and testable.
- Update checklists when process changes are approved.
- Reference checklists from orchestration workflows when relevant.

Core files:
- `pr-checklist.md` for pull request readiness.
- `release-checklist.md` for release preparation.
