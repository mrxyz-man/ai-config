# Project Module

This module stores project context for AI-agent work.

## Goal

Provide enough business and technical context for high-quality planning and implementation decisions.

## Important Files

- `goals.yaml`: measurable business/quality goals and constraints for prioritization.
- `description.md`: what the project does and why it exists.
- `requirements.md`: functional and non-functional requirements.
- `tech-stack.yaml`: current stack and core dependencies.
- `structure.md`: repository map and responsibilities.
- `decisions.md`: key architecture decisions.

## Notes

- Keep goals measurable (metric + target value where possible).
- AI-agent should align trade-offs with `goals.yaml` first.
