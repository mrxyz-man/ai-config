# Skills Module

This module contains reusable AI capabilities that can be routed to roles and task types.

## Included Skills

- `context-hydration.yaml`
- `task-breakdown.yaml`
- `code-generation.yaml`
- `testing.yaml`
- `documentation.yaml`

## Notes

- Skills extend role behavior; they do not replace role contracts.
- Register/enable skills via `registry.yaml`.
- Add project-specific skills under `custom/` and include them in registry.
